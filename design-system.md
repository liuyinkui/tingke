## 1. 设计语言

### 1.1 核心原则

| 原则 | 含义 | 设计表现 |
|------|------|---------|
| **减法优先** | 每个元素存在的前提是：去掉它用户会不会死？ | 首页只有一个 CTA，"开始今日练习" |
| **零思考** | 用户打开 App 不需要想"点什么" | 无 tab bar，无菜单层级 |
| **进度感** | 用户永远知道自己在哪、完成了多少 | 三步进度指示 + 百分比 |

### 1.2 品牌调性

```
专业 · 安静 · 不花哨

像潮汐的安静感 + 多邻国的进度感
去掉潮汐的过于极简 + 去掉多邻国的幼稚感
```

```
    ✅ 进度条填充
    ✅ Streak 数字/火焰
    ✅ 完成页的"完成"状态
    ✅ 当前步骤指示器
    ❌ 不用作普通文字颜色
    ❌ 不用作大面积背景
```

## 7. 动效系统

### 7.1 缓动曲线

```css
--ease-default: cubic-bezier(0.25, 0.1, 0.25, 1)
--ease-spring: cubic-bezier(0.34, 1.56, 0.64, 1)
--ease-exit: cubic-bezier(0.32, 0.72, 0, 1)
```

### 7.2 持续时间

```css
--duration-fast: 150ms   /* hover、按压 */
--duration-normal: 300ms /* 页面过渡 */
--duration-slow: 500ms   /* 完成页庆祝 */
```

### 7.3 动效使用原则

```
✅ 必须做的事：
  · 完成页 🎉 庆祝动画（500ms, spring）
  · Strengthening 打卡数据更新（数字跳动）
  · 听写提交后的对/错标记反馈（闪光 + 变色）
  · Streak 火焰跳动

❌ 绝对不做：
  · 页面切换动画（slide left/right）
  · 卡片飞入飞出
  · 粒子/烟花效果
  · 阻尼滑动
  · 毛玻璃过渡
```

```css
/* 小屏手机 (<375px) */
--mobile-sm: 320px

/* 标准手机 (375px ~ 414px) */
--mobile-md: 375px

/* 大屏手机/小平板 (>414px) */

```

### 11.2 适配策略

```
@优先使用 flex/grid 弹性布局
@间距使用固定值(spacing tokens)，不缩放
@字号不缩放（15px body 在大小屏上一致）
@触摸目标不小于 44px
```

## 13. 品牌 Logo 设计规范

### 13.1 Logo 概念

```
"听刻" = "把声音刻进脑子里"

视觉隐喻：
  声波（听） + 雕刻/印记（刻）

形态：
  一个圆形印章风格
  内部是声波图案 + "听刻"二字
  暗示"刻录"、"印章"、"印记"的概念
```

### 13.2 Logo 规范

```
主 Logo (全称)：
  ┌─────────────────┐
  │   🎵 听刻        │
  │   TINGKE         │
  │  (声波icon)       │
  └─────────────────┘

图标 Logo (仅图标)：
  ┌─────┐
  │ ╱╲╱  │  声波图案，圆形
  │ ╲╱╲  │  类似唱片/印章
  └─────┘

配色：
  · 全色版本：深蓝 #1E3A5F

安全区域：Logo 边缘保留 10% 间距
```
