# 听刻

> 把声音刻进脑子里

一款基于精听法的英语训练App。每天15分钟，盲听→听写→跟读，真正提升英语听力。

## 技术栈

- 前端：React Native (iOS + Android)
- 后端：Node.js + Express
- 数据库：PostgreSQL + Redis
- AI：阿里云语音评测
- 部署：Docker + GitHub Actions

## 项目结构

```
tingke/
├── app/           # React Native 移动端
├── server/        # Node.js 后端 API
├── scripts/       # 素材流水线脚本
└── docs/          # 文档
```
